HTML5-Shooter is a simple shooter game to test out the new features in HTML5 and CSS3.

Update: Moved the game to game.html and game.js. index.html is now a launcher for the game with settings for the user.

Note: This game is only tested on Google Chrome!!!

Controls:
	W - Move player up
	A - Move player left
	S - Move player down
	D - Move player right

	Spacebar - Run

	Left-Click: Shoot
	Right-Click: Scope

Gameplay:
A few targets are created at the beginning of the game which are shootable.
Shooting without the use of the scope results in poors accuracy of shots.
Shooting with the use of the scope results in dead-zero accuracy of shots.
There is a 1 second delay between shots.
